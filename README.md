# TTDH Web Solutions — Agency Website

> **Digital Systems That Scale Businesses**

A world-class, ultra-premium, high-converting agency website for TTDH Web Solutions — a digital agency specialising in web development, AI automation & digital growth.

---

## 🚀 Tech Stack

| Technology | Purpose |
|------------|---------|
| HTML5 | Semantic markup, accessibility |
| CSS3 | Custom properties, glassmorphism, animations |
| JavaScript (ES6+) | Interactivity, animation orchestration |
| [GSAP 3 + ScrollTrigger](https://cdnjs.cloudflare.com/ajax/libs/gsap/3.12.5/gsap.min.js) | Scroll-triggered reveals, counters, parallax, staggered animations |
| [Lenis](https://cdn.jsdelivr.net/npm/@studio-freight/lenis@1.0.42/dist/lenis.min.js) | Buttery smooth scrolling engine |
| [Particles.js](https://cdn.jsdelivr.net/npm/particles.js@2.0.0/particles.min.js) | Animated particle hero background |
| [Vanilla Tilt.js](https://cdn.jsdelivr.net/npm/vanilla-tilt@1.8.1/dist/vanilla-tilt.min.js) | 3D tilt hover effect on service cards |
| [Swiper.js](https://cdn.jsdelivr.net/npm/swiper@11/swiper-bundle.min.js) | Smooth testimonial carousel |
| [Google Fonts](https://fonts.google.com/) | Inter + Space Grotesk premium typography |

---

## ✨ Features

- **Dark luxury theme** — deep charcoal/black with electric blue & violet gradient accents
- **Glassmorphism panels** with soft glow borders on all cards
- **Sticky navbar** — transparent to glass blur on scroll
- **Full-screen mobile menu** with hamburger toggle
- **Hero section** with staggered word-by-word reveal animation
- **Particles.js** animated background in hero
- **3D tilt hover cards** on all service cards (Vanilla Tilt)
- **Magnetic CTA buttons** that subtly pull toward cursor
- **Mouse-follow glow effect** — radial gradient follows the cursor
- **Animated counters** — count up to target when scrolled into view
- **Scroll-triggered animations** on every section (GSAP ScrollTrigger)
- **Parallax** glow orbs in hero
- **Swiper.js testimonial carousel** with autoplay & touch support
- **Animated process timeline** with gradient line draw on scroll
- **Before/After comparison** cards in results section
- **Contact form** with styled validation & submit feedback
- **Floating WhatsApp button** with pulse animation & tooltip
- **Fully responsive** — desktop, tablet & mobile
- **Accessible** — semantic HTML5, ARIA labels, keyboard navigable
- **SEO meta tags** — title, description, Open Graph

## ▶️ How to Run

Simply open `index.html` in any modern browser — no build step, server, or npm required.

```bash
# macOS / Linux
open index.html

# Windows
start index.html
```

> All libraries are loaded via CDN. An internet connection is required for fonts and CDN assets on first load.
